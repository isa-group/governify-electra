How to load a custom scenario

  1. Model the structure of the APIs using the OpenAPI Specification. Next, model the usage limitations in every service by using the SLA4OAI Specification.
  2. Publish each of these files with a URL. For instance, Github Gist service can be useful ( see an example).
  3. Write the X-Conversation document in the editor below (an initial example is provided). Then, save the model.
  4. Click in the calculate usage limitations button.
