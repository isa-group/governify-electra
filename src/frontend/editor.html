How to load a custom scenario

  1. Model the structure of the APIs using the OpenAPI Specification. Next, model the usage limitations in every service by using the SLA4OAI Specification.
  2. Publish each of these files with an URL. For instance, Github Gist service can be useful (see an example).
  3. Write the x-conversation document in the editor below (an initial example is provided).
  4. Click in the save and calculate induced usage limitations button to calculate the induced limitations.

How to use the example scenario

  1. A model the structure of the APIs using the OpenAPI Specification is provided. Also, the usage limitations in every service by using the SLA4OAI Specification are included.
  2. Make changes in the x-conversation document in the editor below.
  3. Click in the save and calculate induced usage limitations button to calculate the induced limitations.
